Para compilar o programa, podemos usar o seguinte comando make\+:


\begin{DoxyItemize}
\item \textquotesingle{}make\textquotesingle{}
\end{DoxyItemize}\hypertarget{index_autotoc_md5}{}\doxysubsubsection{Opções de execução}\label{index_autotoc_md5}
O Makefile oferece diferentes opções que podem ser utilizadas para executar o programa\+:


\begin{DoxyItemize}
\item \textquotesingle{}run2f\textquotesingle{}\+: A opção run2f é utilizada para executar o programa sem nenhuma opção extra com apenas dois ficheiros passados por argumento com os dados separados por ponto e vírgulas (;)
\item \textquotesingle{}run3f\textquotesingle{}\+: A opção run2f é utilizada para executar o programa sem nenhuma opção extra com três ficheiros passados por argumento com os dados separados por ponto e vírgulas (;)
\item \textquotesingle{}ajuda\textquotesingle{}\+: A opção ajuda é utilizada para fornecer ajuda ao utilizador sobre os comandos a utilizar
\item \textquotesingle{}tab2f\textquotesingle{}\+: A opção tab2f é utilizada para executar o programa com apenas dois ficheiros passados por argumento com os dados separados por tabs
\item \textquotesingle{}tab3f\textquotesingle{}\+: A opção tab3f é utilizada para executar o programa com três ficheiros passados por argumento com os dados separados por tabs
\end{DoxyItemize}\hypertarget{index_autotoc_md6}{}\doxysubsubsection{Comandos de execução}\label{index_autotoc_md6}
Para executar o programa utilizamos os seguintes comandos\+:


\begin{DoxyItemize}
\item \textquotesingle{}make run2f\textquotesingle{}\+: Compila e executa o programa com a opção \textquotesingle{}run2f\textquotesingle{}
\item \textquotesingle{}make run3f\textquotesingle{}\+: Compila e executa o programa com a opção \textquotesingle{}run3f\textquotesingle{}
\item \textquotesingle{}make ajuda\textquotesingle{}\+: Compila e executa o programa com a opção \textquotesingle{}ajuda\textquotesingle{}
\item \textquotesingle{}make tab2f\textquotesingle{}\+: Compila e executa o programa com a opção \textquotesingle{}tab2f\textquotesingle{}
\item \textquotesingle{}make tab3f\textquotesingle{}\+: Compila e executa o programa com a opção \textquotesingle{}tab3f\textquotesingle{}
\end{DoxyItemize}\hypertarget{index_autotoc_md7}{}\doxysubsection{Divisão de Tarefas}\label{index_autotoc_md7}
Durante a execução do projeto, embora tenhamos colaborado de forma conjunta em várias etapas e nos encontrássemos envolvidos em todas as partes do trabalho, identificamos uma distribuição específica de responsabilidades. Cada membro do grupo concentrou-\/se em áreas específicas, resultando numa colaboração eficiente. As principais responsabilidades de cada membro foram\+:


\begin{DoxyItemize}
\item Pedro Ribeiro (27960)\+: Manipulação do código para a possibilidade de inserir dados por argumentos e por stdin;

\item Carolina Branco (27983)\+: Configuração e criação da documentação Doxygen e criação do documento refman.\+pdf.
\end{DoxyItemize}

Além de todas estas funções, cada um de nós participou igualmente na realização do relatório, cada um na sua responsabilidade correspondente.
